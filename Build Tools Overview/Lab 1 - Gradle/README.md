# Lab 1 - Building and Packaging Java Application using Gradle

## 📖 Overview

This lab demonstrates how to build, test, package, and run a Java calculator application using **Gradle** on **Red Hat Enterprise Linux (RHEL)**.

---

## 🎯 Objective

- Install and verify Gradle.
- Build a Java project.
- Run unit tests.
- Generate the executable JAR file.
- Execute the application.

---

## 🖥️ Environment

- Operating System: Red Hat Enterprise Linux (RHEL)
- Java: JDK 17
- Build Tool: Gradle

---

## 📁 Directory Structure

```text
Source Code/
└── calculator-gradle
    ├── build.gradle
    ├── settings.gradle
    └── src
```
