# Gradle Commands

## Verify Gradle Installation

```bash
gradle --version
```

---

## Clone Repository

```bash
git clone https://github.com/Ibrahim-Adel15/calculator-gradle.git
```

---

## Navigate to Project

```bash
cd calculator-gradle
```

---

## Run Unit Tests

```bash
gradle test
```

---

## Build the Project

```bash
gradle build
```

---

## Verify Generated JAR

```bash
ls build/libs
```

---

## Run the Application

```bash
java -jar build/libs/calculator.jar
```
