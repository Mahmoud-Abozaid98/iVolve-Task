# Maven Commands

## Verify Maven Installation

```bash
mvn --version
```

---

## Clone Repository

```bash
git clone https://github.com/Ibrahim-Adel15/calculator-maven.git
```

---

## Navigate to Project

```bash
cd calculator-maven
```

---

## Run Unit Tests

```bash
mvn test
```

---

## Build the Project

```bash
mvn clean package
```

---

## Verify Generated JAR

```bash
ls target
```

---

## Run the Application

```bash
java -jar target/calculator.jar
```
