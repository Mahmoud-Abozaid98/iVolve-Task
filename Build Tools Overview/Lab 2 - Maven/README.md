# Lab 2 - Building and Packaging Java Application using Maven

## 📖 Overview

This lab demonstrates how to build, test, package, and run a Java calculator application using **Maven** on **Red Hat Enterprise Linux (RHEL)**.

---

## 🎯 Objective

- Install and verify Maven.
- Build a Java project.
- Run unit tests.
- Generate the executable JAR file.
- Execute the application.

---

## 🖥️ Environment

- Operating System: Red Hat Enterprise Linux (RHEL)
- Java: JDK 17
- Build Tool: Maven
